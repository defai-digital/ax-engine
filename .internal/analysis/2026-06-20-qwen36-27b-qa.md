## Bug Fixed

The server preset list did not expose a `qwen3.6-27b` preset, even though local Qwen 3.6 27B snapshots were available and worked when launched through explicit `--model-id qwen36-27b`.

Fix:

- Added `qwen3.6-27b` / `qwen36-27b` as an MLX preview server preset.
- Accepted the cached `mlx-community/Qwen3.6-27B-4bit` snapshot with manifest model type `qwen3_5`.
- Added preset render and session config regression coverage.
